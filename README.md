# Custom user scripts for Tampermonkey or similar extensions

## Jira: Copy issue number instead of link

This script allows you to copy the issue number instead of the link when you click the copy button in Jira.\
Just press the option (alt) key to toggle the behavior. When the option key is pressed while clicking the button, the issue number will be copied to the clipboard instead of the link.
